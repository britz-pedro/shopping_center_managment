# Shopping Center Management System (Java)

## Description
Academic Java project developed during college to practice core programming and
Object-Oriented Programming (OOP) concepts. The project simulates a shopping center
domain and is organized in stages that reflect the learning process.

## Technologies
- Java
- Visual Studio Code (VS Code)

## Concepts Practiced
- Classes and objects
- Constructors and encapsulation
- Association between classes
- Inheritance and polymorphism
- Arrays and data handling
- Logical problem-solving

## Project Structure
- **Etapa 1 – Criação de classes**  
  Definition of core domain classes and attributes.

- **Etapa 2 – Associação entre classes**  
  Relationships between objects and main execution logic.

- **Etapa 3 – Herança e Polimorfismo**  
  Use of inheritance to model different store and product types.

- **Etapa 4 – Array**  
  Practice with arrays to store and manage collections of objects.

## Notes
This project was developed for learning purposes and focuses on Java fundamentals
and code organization.
